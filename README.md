# Gaia Seven

Gaia Protocol Discord Bot

## Get Started

```
npm install
npm run dev
```

## .env

```
PORT=3000
DISCORD_TOKEN=
DISCORD_CLIENT_ID=
DISCORD_GUILD_ID=
```

## Structure

```
src
- config
- loaders: settings modules
- services: business logics
- routes
index.ts: Entry point
tests
.env
```
